## Installation
```
$ git clone https://github.com/cloraronco/42_Push_swap
$ cd 42_Push_swap/
$ make
$ git clone https://github.com/o-reo/push_swap_visualizer.git
$ cd push_swap_visualizer/
$ mkdir build/
$ cd build/
$ cmake ..
$ make
$ ./bin/visualizer
```

## Play
